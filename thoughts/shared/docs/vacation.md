# Vacation (/vacation)

## Purpose
Show vacation balance, carryover, upcoming vacations, and transaction history for a selected year.

## Audience
- Users: View their own vacation balance.
- Admins: Same view for their own employee profile.

## How to use (step-by-step)
1. Open Vacation from the main navigation.
2. Select the year using the year selector.
3. Review Balance Breakdown and carryover warnings (current year).
4. Review Upcoming Vacation (current year).
5. Review Transaction History for accruals and deductions.

## Data and fields
- Base entitlement, additional entitlement, carryover, manual adjustments.
- Used days, planned days, remaining days.
- Carryover expiration date (if present).

## Where changes take effect / integrations
- Absence requests that affect vacation adjust planned/used values.
- Year Overview uses the same vacation balance data.

## Troubleshooting
- "No vacation data" indicates no balance has been calculated for the selected year.
- If balances look wrong, verify absence types and tariff vacation settings.
