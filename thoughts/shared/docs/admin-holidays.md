# Holidays (Admin) (/admin/holidays)

## Purpose
Manage holiday dates that affect daily calculations and absence working-day counts.

## Audience
- Admins: Full access.
- Users: Not available; non-admins are redirected to the Dashboard.

## How to use (step-by-step)
1. Open Holidays from the Admin navigation.
2. Click New Holiday and select a date.
3. Enter a name and choose full/half day.
4. Choose whether it applies to all departments or a specific department.
5. Save and use Edit to update if needed.

## Data and fields
- Date, name.
- Half-day flag.
- Applies to all vs department-specific.

## Where changes take effect / integrations
- Holidays appear in Timesheet and Absences calendars.
- Holiday flags affect daily calculations and vacation impact previews.

## Troubleshooting
- Date and name are required.
- If not applying to all, a department must be selected.
