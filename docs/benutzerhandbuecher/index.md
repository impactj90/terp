## Schnelleinstieg

### Für neue Mitarbeiter
1. Starten Sie mit dem [Dashboard](./dashboard.md)
2. Lernen Sie die [Stempeluhr](./stempeluhr.md) kennen
3. Verstehen Sie den [Stundenzettel](./stundenzettel.md)
4. Beantragen Sie [Abwesenheiten](./abwesenheiten.md)

### Für Teamleiter
1. Nutzen Sie die [Teamübersicht](./teamuebersicht.md)
2. Bearbeiten Sie [Genehmigungen](./genehmigungen.md)

### Für Administratoren
1. Beginnen Sie mit [Mitarbeiter](./mitarbeiter.md)
2. Konfigurieren Sie [Tagespläne](./tagesplaene.md) und [Wochenpläne](./wochenplaene.md)
3. Richten Sie [Abwesenheitsarten](./abwesenheitsarten.md) ein
4. Prüfen Sie [Einstellungen](./einstellungen.md)

## Hilfe & Support

Bei Problemen oder Fragen:
1. Prüfen Sie den Abschnitt "Problembehandlung" im jeweiligen Handbuch
2. Kontaktieren Sie Ihren Systemadministrator
3. Schauen Sie in die [Audit-Protokolle](./audit-protokolle.md) für Fehleranalyse
