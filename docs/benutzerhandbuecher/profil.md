# Profil - Benutzerhandbuch

## Überblick

Die **Profilseite** zeigt Ihre persönlichen Daten, Beschäftigungsinformationen und ermöglicht die Verwaltung Ihrer Kontoeinstellungen. Hier sehen Sie alle relevanten Informationen, die im System über Sie gespeichert sind, und können bestimmte Einstellungen selbst ändern.

**Wer nutzt dieses Modul:**
- Alle Mitarbeiter zur Einsicht ihrer persönlichen Daten
- Für die Änderung von Kontoeinstellungen (z.B. Passwort)
- Zur Prüfung der Beschäftigungsdaten

## Voraussetzungen

1. **Benutzerkonto**: Sie müssen im System angemeldet sein.
2. **Mitarbeiterdatensatz** (optional): Für vollständige Profildaten sollte Ihr Konto mit einem Mitarbeiterdatensatz verknüpft sein.

## Zugang zum Modul

**Navigationspfad:** Hauptmenü → Profil (oder: Benutzermenü oben rechts → Profil)

**Mobil:** Über das Seitenmenü oder Benutzer-Avatar

**Direkte URL:** `/profile`

## Funktionen & Bedienelemente

### 1. Profilheader
Zeigt Ihr Profilbild und grundlegende Informationen:
- **Avatar**: Ihr Profilbild oder Initialen
- **Name**: Ihr vollständiger Name
- **Position**: Ihre Stellenbezeichnung
- **E-Mail**: Ihre E-Mail-Adresse

### 2. Persönliche Informationen (Personal Info Card)
Ihre persönlichen Daten:

| Feld | Beschreibung |
|------|--------------|
| **Vorname** | Ihr Vorname |
| **Nachname** | Ihr Nachname |
| **E-Mail** | Ihre E-Mail-Adresse |
| **Telefon** | Ihre Telefonnummer |
| **Geburtsdatum** | Ihr Geburtsdatum |
| **Adresse** | Ihre Anschrift |

### 3. Beschäftigungsdetails (Employment Details Card)
Informationen zu Ihrem Arbeitsverhältnis:

| Feld | Beschreibung |
|------|--------------|
| **Personalnummer** | Ihre eindeutige Kennung |
| **Eintrittsdatum** | Beginn Ihres Arbeitsverhältnisses |
| **Beschäftigungsart** | Vollzeit, Teilzeit, etc. |
| **Abteilung** | Ihre Organisationseinheit |
| **Team** | Ihr zugeordnetes Team |
| **Kostenstelle** | Ihre Kostenstelle |
| **Standort** | Ihr Arbeitsstandort |
| **Vorgesetzter** | Ihr direkter Vorgesetzter |

### 4. Notfallkontakte (Emergency Contacts Card)
Liste Ihrer hinterlegten Notfallkontakte:
- Name des Kontakts
- Beziehung (Ehepartner, Eltern, etc.)
- Telefonnummer
- Priorität

### 5. Zugangskarten (Access Cards Card)
Ihre Chipkarten/Zugangsmittel:
- Kartennummer
- Typ (Mitarbeiterausweis, Parkausweis)
- Gültigkeitszeitraum
- Status (aktiv/gesperrt)

### 6. Zeitplan (Time Plan Card)
Ihr aktueller Arbeitszeitplan:
- Zugewiesener Wochenplan
- Tägliche Sollstunden
- Gültig ab Datum

### 7. Kontoeinstellungen (Account Settings Card)
Verwaltung Ihrer Benutzereinstellungen:
- **E-Mail ändern**: Ihre Login-E-Mail anpassen
- **Passwort ändern**: Neues Passwort setzen
- **Sprache**: Bevorzugte Sprache der Oberfläche
- **Zeitzone**: Ihre Zeitzone

## Schritt-für-Schritt Anleitungen

### Persönliche Daten einsehen

1. Navigieren Sie zu **Profil**
2. Ihre Daten werden automatisch angezeigt
3. Prüfen Sie die verschiedenen Karten für unterschiedliche Informationen

### Passwort ändern

1. Öffnen Sie die **Profilseite**
2. Scrollen Sie zu **Kontoeinstellungen**
3. Klicken Sie auf **"Passwort ändern"**
4. Geben Sie Ihr **aktuelles Passwort** ein
5. Geben Sie Ihr **neues Passwort** ein
6. Bestätigen Sie das neue Passwort
7. Klicken Sie auf **"Speichern"**

### Spracheinstellung ändern

1. Öffnen Sie die **Profilseite**
2. Gehen Sie zu **Kontoeinstellungen**
3. Wählen Sie Ihre bevorzugte **Sprache** aus dem Dropdown
4. Die Änderung wird sofort übernommen

### Notfallkontakt prüfen

1. Öffnen Sie die **Profilseite**
2. Scrollen Sie zu **Notfallkontakte**
3. Prüfen Sie die hinterlegten Kontakte auf Aktualität
4. Bei Änderungsbedarf kontaktieren Sie die Personalabteilung

### Zeitplan einsehen

1. Öffnen Sie die **Profilseite**
2. Gehen Sie zur Karte **Zeitplan**
3. Sehen Sie Ihren zugewiesenen Wochenplan
4. Prüfen Sie die täglichen Sollstunden

### Zugangskarten prüfen

1. Öffnen Sie die **Profilseite**
2. Scrollen Sie zu **Zugangskarten**
3. Prüfen Sie Kartennummer und Status
4. Bei Problemen kontaktieren Sie die IT oder Personalabteilung

### Fehlerhafte Daten melden

1. Prüfen Sie Ihre Daten auf der Profilseite
2. Notieren Sie sich die fehlerhaften Informationen
3. Kontaktieren Sie Ihren Administrator oder die Personalabteilung
4. Geben Sie an, welche Daten korrigiert werden müssen

## Tipps & Best Practices

1. **Regelmäßig prüfen**: Kontrollieren Sie Ihre Daten periodisch auf Richtigkeit.

2. **Notfallkontakte aktuell halten**: Informieren Sie die Personalabteilung bei Änderungen.

3. **Sicheres Passwort**: Wählen Sie ein starkes, einzigartiges Passwort.

4. **Passwort regelmäßig ändern**: Ändern Sie Ihr Passwort in regelmäßigen Abständen.

5. **Zugangskarten melden**: Melden Sie verlorene oder defekte Zugangskarten sofort.

6. **Zeitplan verstehen**: Kennen Sie Ihren Zeitplan für die korrekte Sollzeitberechnung.

## Problembehandlung

### Profil zeigt eingeschränkte Ansicht
**Ursache**: Kein Mitarbeiterdatensatz mit Ihrem Benutzerkonto verknüpft.
**Lösung**: Kontaktieren Sie Ihren Administrator.

### Passwortänderung schlägt fehl
**Ursache**: Aktuelles Passwort falsch oder neues Passwort erfüllt Anforderungen nicht.
**Lösung**: Prüfen Sie das aktuelle Passwort. Das neue Passwort muss ggf. Mindestanforderungen erfüllen (Länge, Komplexität).

### Daten können nicht bearbeitet werden
**Ursache**: Die meisten Profildaten können nur von Administratoren geändert werden.
**Lösung**: Kontaktieren Sie Ihre Personalabteilung für Datenänderungen.

### Zeitplan erscheint falsch
**Ursache**: Falscher Wochenplan zugewiesen oder kürzliche Änderung noch nicht aktiv.
**Lösung**: Kontaktieren Sie Ihren Administrator.

### Notfallkontakte können nicht hinzugefügt werden
**Ursache**: Diese Funktion muss von Administratoren durchgeführt werden.
**Lösung**: Wenden Sie sich an die Personalabteilung.

### Zugangskarte zeigt falschen Status
**Ursache**: Statusaktualisierung noch nicht durchgeführt.
**Lösung**: Kontaktieren Sie die IT-Abteilung.

### Sprachänderung wird nicht übernommen
**Ursache**: Browser-Cache oder Cookies.
**Lösung**: Aktualisieren Sie die Seite oder löschen Sie den Browser-Cache.
